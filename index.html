  • Roteadores decidem caminhos.
  • Switches organizam conversas.
    • 1. Estrutura de Diretórios (Hierarquia)
  • / → Raiz do sistema (tudo começa aqui)
  • /home → Pasta dos usuários
  • /bin → Comandos essenciais
  • /etc → Arquivos de configuração
  • /usr → Programas instalados
  • /var → Logs e arquivos variáveis
  • /tmp → Arquivos temporários
    • 2. Navegação na Árvore
  • pwd → Mostra onde você está
  • ls → Lista arquivos
  • ls -l → Lista detalhada
  • ls -a → Mostra arquivos ocultos
  • cd pasta → Entra na pasta
  • cd .. → Volta um nível
  • cd / → Vai para a raiz
  • cd ~ → Vai para sua home
    • 3. Manipulação de Arquivos
  • touch arquivo.txt → Cria arquivo vazio
  • mkdir pasta → Cria diretório
  • rm arquivo.txt → Remove arquivo
  • rm -r pasta → Remove pasta
  • cp origem destino → Copia arquivo
  • mv origem destino → Move ou renomeia
  • cat arquivo.txt → Mostra conteúdo
  • less arquivo.txt → Visualiza paginado
  • head arquivo.txt → Primeiras linhas
  • tail arquivo.txt → Últimas linhas
    • 4. Permissões
  • chmod 755 arquivo → Muda permissões
  • chown usuario arquivo → Muda dono
  • ls -l → Mostra permissões
    • 5. Processos
  • ps → Lista processos
  • top → Monitor em tempo real
  • htop → Monitor avançado
  • kill PID → Encerra processo
    • 6. Rede
  • ping site.com → Testa conexão
  • ip a → Mostra IP
  • curl url → Faz requisição web
  • wget url → Baixa arquivo
    • 7. Pacotes (Debian/Ubuntu)
  • sudo apt update → Atualiza lista
  • sudo apt upgrade → Atualiza sistema
  • sudo apt install pacote → Instala
  • sudo apt remove pacote → Remove
    • 8. Variáveis e Sistema
  • echo $PATH → Mostra caminhos de comandos
  • which comando → Mostra localização
  • history → Histórico de comandos
  • clear → Limpa terminal
  • man comando → Manual
    • 9. Compactação
  • tar -cvf arquivo.tar pasta → Compacta
  • tar -xvf arquivo.tar → Extrai
  • zip arquivo.zip pasta → Compacta
  • unzip arquivo.zip → Extrai
    • Regra Mental Final
  • ✔ Tudo começa em /
  • ✔ Pense sempre: "Onde estou?" → pwd
  • ✔ Liste antes de agir → ls
  • ✔ Teste na sua /home

IP Decimal → Binário

Converte um endereço IPv4 comum para binário.

Dica: cada octeto vira 8 bits (0–255).

IP Binário → Decimal

Converte IPv4 binário para formato decimal.

Dica: 8 bits por octeto, separados por ponto.
